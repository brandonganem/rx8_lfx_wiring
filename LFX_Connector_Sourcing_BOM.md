## Sourcing Tips
*   **Mouser / DigiKey / Waytek Wire:** Search using the "Housing P/N" (e.g., 13585860). You will also need to order the matching terminals and seals separately if buying housings.
*   **RockAuto / Amazon / eBay:** Search using the "Pigtail P/N" (e.g., PT2345) for a pre-wired connector pigtail. This is often easier but more expensive per connector.
*   **Ballenger Motorsports / CBM Motorsports:** Specialty retailers often sell complete connector kits for GM engines.
